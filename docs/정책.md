# 대시보드

## 1. 관리자 대시보드

**표시 항목**

- 일 매출, 월 매출, 일 방문자, 이번주 방문자
- 일별 매출 요약 / 월별 매출요약 / 이번달 매출 구성
- 최근 7일간 방문자 추이

## 2. 트레이너 대시보드

**표시항목** 

- 나의 총 PT 회원 수, 나의 총 잔여 PT 수, 오늘 예약된 PT 수, 오늘 예약된 상담 수
- 나의 회원 목록
    - 회원 정보 (사진/이름/회원번호), 남은 PT 횟수 및 만료일, 잔여 헬스권 일수 및 만료일, 출석률
- 공지사항

# 회원 상세

## **회원 상세 조회**
### 1) 노출 정보
- 회원 정보 
    - 사진
    - 이름
    - 회원번호
    - 회원 구분 (헬스권, PT)
    - 생년월일 및 만나이
    - 성별
    - 핸드폰 번호
    - 주소
    - 문자 수신 동의 여부
    - 출석률
    - 이용 가능 지점
        - ``` 다지점 이용 가능 회원권의 경우 여러개 지점이 나올 수 있다. ```
    - 잔여 미수금 **(필요 여부 확인)**
    - 방문 / 가입 경로 **(필요 여부 확인)**
    - 특이사항 메모
    - **정지 등록 버튼**
        - 정지 등록 버튼을 클릭 시 
    - **회원 수정 버튼**    
    
- 헬스장 이용권 정보
    - 노출 정보
        - 상품명 (ex) 헬스 이용권 1개월권, 헬스 이용권 12개월권) 
        - 남은 기간 (N일), 상품 사용 기간 (25.3.27 ~ 26.3.27)
        - 이용 가능 지점
        - **결제 등록 버튼 (관리자 only)**
    - 만약 헬스장 이용권이 없으면, 헬스장 이용권 정보가 없습니다 라는 문구를 보여줘야 한다.
    - 결제 등록 버튼을 클릭 시, 결제 등록 팝업이 뜬다. 단 이때 일반 직원은 버튼이 비활성화 되어야 한다.
- PT 레슨권 정보
    - 노출 정보
        - 상품명 (ex) PT 10회권, PT 20회) 
        - 담당 트레이너 정보
        - 남은 횟수 (N회), 상품 사용 기간 (25.3.27 ~ 26.3.27)
        - **결제 등록 버튼 (관리자 only)**
    - 만약 PT 이용권이 없으면, PT 이용권 정보가 없습니다 라는 문구를 보여줘야 한다.
- 락커 이용권
    - 노출 정보
        - 상품명
        - 남은 기간 (N일), 상품 사용 기간 (25.3.27 ~ 26.3.27)
        - 락커 정보
        - **결제 등록 버튼 (관리자 only)**
        - 락커 특이사항 메모
    - 만약 락커 이용권이 없으면, 락커 이용권 정보가 없습니다 라는 문구를 보여줘야 한다.
- 기타 상품
    - 노출 정보
        - 상품명 (Ex) 식음료, 공간대여, 촬영협조 등등)
        - 상품 사용 기간 (25.3.27 ~ 26.3.27)
        - **결제 등록 버튼 (관리자 only)**
    - 만약 운동용품이 없으면, 운동용품 정보가 없습니다 라는 문구를 보여줘야 한다.
- 출석 현황
    - 표시 내용
        - 출석률
        - 출석 캘린더를 보여준다. (디자인처리)
        - 출석 기록 테이블 (날짜, 출석 시간, 출석 여부) 
    - 출석 캘린더에는 출석한 날짜와 미출석한 날짜를 다른 색깔로 구분하여 보여준다
    - 출석 기록 테이블의 출석기록은 최초에 10개까지만 보여주고, 그 이상 있다면 페이징 처리해서 보여준다

- 결제 내역
    - 표시 내용
        - 결제일시
        - 상품 시작일시
        - 상품 구분 (헬스장 이용권 / PT 레슨권 / 운동 용품)
        - 상품명 (ex)
        - 결제 상태 (결제 완료 / 대기 / 취소)
    - 특정 레코드를 클릭하면 결제 상세 팝업을 띄워준다. 
    - 최초에 10개까지만 보여주고, 그 이상 있다면 페이징 처리해서 보여준다
    
- 정지 기록 **(필요 여부 확인 필요)**
    - 표시 내용 
        - 정지 일시 
        - 정지 기간 
        - 정지 사유
        - 승인 담당자 
    - 정지 기록은 최초에 10개까지만 보여주고, 그 이상 있다면 페이징 처리해서 보여준다

### 2) 회원 수정 팝업 (관리자 only)
- 수정할 수 있는 정보 
    - 이름
    - 생년월일 
    - 성별
    - 휴대폰번호
    - 주소
    - 최초 가입일
    - 가입 경로
    - 회원 메모
- 하단에 돌아가기, 저장 버튼 추가

### 3) 결제 등록 팝업 (관리자 only)

- 구매 상품
- 시작일
- 결제 직원
- 담당 트레이너
- 결제 정보 (결제 일시, 수단)
- 마케팅 관련 정보 (목적, 방문 경로)
- 상품 금액
    - 등록된 상품 금액
    - 실제 결제 금액
    - 결제 직원 매출 실적
    - 미수금

### 4) 회원권 정지 팝업 (관리자 only)
- 회원권 만료일 노출 
- 정지 기간 입력 (5일, 10일 등등 수기 입력)
- 취소, 등록

### 5) 결제 상세 팝업
- 노출 정보
    - 상품 정보
        - 상품 구분 (헬스장 이용권 / PT 레슨권 / 운동 용품)
        - 상품명
        - 상품 시작일 
        - 상품 종료일
        - 상품 담당자 매출 실적
        - 담당 직원 
    - 결제 정보
        - 결제일시
        - 결제 수단
        - 상품 금액 
        - 실제 결제 금액
        - 상품 담당자 매출 실적
        - 미수금
        - 결제 상태 (결제 완료 / 대기 / 취소)
    - 수정 / 삭제 버튼 
        - 관리자에게만 활성화되고, 일반 직원은 보이지 않아야한다.

- 수정 버튼 클릭 시 현재 팝업에서 각 요소가 수정할 수 있는 상태로 변경되고, 하단에 취소 / 저장 버튼이 노출된다.
- 삭제 버튼 클릭 시 삭제 확인 alert 팝업을 한번 띄워주고, 삭제 버튼 클릭 시 해당 결제 내역을 삭제한다.
